테스트용 HTML 인덱스 페이지

기본 구조와 뼈대만 있는 매우 단순한 웹 페이지입니다.

소개 주요 내용 다른 페이지 외부 링크 (Google)

소개

이 문서는 **HTML5** 표준에 맞춰 작성된 간단한 인덱스 파일입니다. 다른 개발 환경이나 프레임워크 없이 순수 HTML과 최소한의 CSS (인라인 스타일)로만 구성되어 있습니다.

개발 초기 단계의 **화면 테스트** 또는 **링크 연결 확인** 용도로 사용하기 적합합니다.

주요 내용

  • **제목 태그 (h1, h2):** 페이지의 구조와 계층을 나타냅니다.
  • **단락 태그 (p):** 일반 텍스트 내용을 담습니다.
  • **목록 태그 (ul/li):** 정보를 구조화하여 나열합니다.
  • **링크 태그 (a):** 다른 페이지나 외부 URL로 이동합니다.
  • **이미지 태그 (img):** 아래에 표시됩니다.

이미지 예시

이미지 태그는 `src` 속성으로 경로를 지정합니다. 현재는 가상의 경로를 사용했습니다.
